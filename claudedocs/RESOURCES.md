## Scaffold Gradle Commands

```bash
./gradlew build          # Compile player code
./gradlew run            # Run game with gradle.properties settings
./gradlew update         # Update to latest version (run frequently!)
./gradlew zipForSubmit   # Create submission zip
./gradlew tasks          # List all available tasks
./gradlew headless       # Run headless match (engine repo)
```
